# 🔄 SIPOC — Visão macro de processo

**S**uppliers → **I**nputs → **P**rocess → **O**utputs → **C**ustomers

## Template

| Suppliers (Fornecedores) | Inputs (Entradas) | Process (Processo) | Outputs (Saídas) | Customers (Clientes) |
| :--- | :--- | :--- | :--- | :--- |
| Banco X | Extrato OFX | 1. Importar<br>2. Validar<br>3. Conciliar<br>4. Aprovar | Relatório conciliação | Contabilidade |
| ERP SAP | Lançamentos contábeis | | Trilha auditoria | Auditoria |
| Área comercial | Notas fiscais | | Alerta e-mail | Gestor Financeiro |

## Quando usar
- **Início do projeto** — enxergar o processo "de fora"
- **Escopo do BRD** — decidir o que entra e sai
- **Workshops** — alinhar visão macro antes de mergulhar em detalhes

## Regra prática
Um SIPOC bem feito **cabe em uma folha A4** e responde: *"o que este processo transforma e para quem?"*
