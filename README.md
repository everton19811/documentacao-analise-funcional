# 📋 Documentação de Análise Funcional & Engenharia de Requisitos

Este repositório reúne modelos práticos, artefatos funcionais e boas práticas de Engenharia de Requisitos, Mapeamento de Processos (BPMN) e Governança de Sistemas. O objetivo é demonstrar a estruturação de entregáveis padrão para projetos corporativos de software.

---

## 🛠️ Tecnologias & Ferramentas Utilizadas
- **Modelagem de Processos:** BPMN 2.0 (Excalidraw, Figma, Bizagi)
- **Gestão Ágil & Requisitos:** User Stories, Critérios de Aceite (BDD/Gherkin), Matriz de Rastreabilidade
- **Metodologias:** Scrum, Kanban, ITIL v4, PMBOK
- **Sistemas de Referência:** ERPs, Sistemas de Integradores (Oracle / IBM Cognos TM1)
