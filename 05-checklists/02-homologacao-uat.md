# ✅ Checklist — Homologação (UAT)

## Antes da UAT
- [ ] Ambiente HML atualizado com última build
- [ ] Massa de dados carregada e anonimizada
- [ ] Usuários de teste criados nos perfis certos
- [ ] Casos de teste escritos e revisados
- [ ] Key user avisado 5 dias antes
- [ ] Treinamento executado (se aplicável)

## Durante a UAT
- [ ] AF acompanha cada sessão
- [ ] Cada CT executado tem evidência (print, log)
- [ ] Bugs registrados no Jira com severidade
- [ ] Bugs bloqueantes vão para retest imediato

## Ao final da UAT
- [ ] Todos os CT dos RF "Must" passaram
- [ ] Zero bugs críticos abertos
- [ ] Bugs "Alto" ≤ limite acordado
- [ ] RTM atualizada
- [ ] Termo de aceite assinado (key user, PO, sponsor)
- [ ] Auditor externo (se aplicável) validou compliance
- [ ] Plano de go-live acionado
